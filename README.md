# موقع البورصة - أسعار الدواجن والبيض

موقع ويب شامل لعرض أسعار الدواجن والبيض والأعلاف والخامات في السوق المصري.

## المميزات

- **واجهة عربية كاملة** مع دعم الاتجاه من اليمين لليسار (RTL)
- **5 أقسام رئيسية:**
  - الخامات (ذرة، صويا، نخالة، إلخ)
  - الأعلاف (شركات مختلفة)
  - البيض (أبيض، أحمر، بلدي)
  - الكتاكيت (أنواع مختلفة)
  - الدواجن (فراخ بيضاء، ساسو، رومي، إلخ)

- **معلومات المستخدم:**
  - صورة شخصية
  - اسم المستخدم
  - إعدادات قابلة للتخصيص

- **تحديث التاريخ تلقائياً**
- **تصميم متجاوب** يعمل على جميع الأجهزة
- **حفظ البيانات محلياً** باستخدام localStorage

## كيفية الاستخدام

1. افتح ملف `index.html` في متصفح الويب
2. استخدم الأزرار العلوية للتنقل بين الأقسام المختلفة
3. اضغط على زر "الإعدادات" لتعديل معلوماتك الشخصية
4. البيانات تُحفظ تلقائياً في المتصفح

## الملفات

- `index.html` - الصفحة الرئيسية
- `styles.css` - ملف التنسيقات
- `script.js` - الوظائف التفاعلية

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

## التطوير المستقبلي

- إضافة صفحات تفصيلية لكل شركة أعلاف
- إضافة نظام تحديث الأسعار تلقائياً
- إضافة رسوم بيانية لتتبع تغيرات الأسعار
- إضافة نظام تنبيهات للأسعار
